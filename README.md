# gnu-gesta

Application web de gestion des stages.

## Prérequis

- Node.js 20+

## Lancer en développement

**Backend:**
```bash
cd backend
npm install
npm run dev
```

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```

## Tests

```bash
cd backend && npm test
```

## Build frontend

```bash
cd frontend && npm run build
```
